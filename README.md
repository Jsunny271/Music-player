# Music-player
Music player in java to play mp3 songs.
This player consist of various buttons like next,back,play,pause,etc.
It consist of various flag values.
It also uses multithreading to perform various tasks at a time.
For every botton there is different event or action to perform specific operations.
It can also choose the file with the help of JFileChooser.
It also have auto play and loop option for user to access easily and effectively.
All work done in java with the help of oop concept and multithreading.
